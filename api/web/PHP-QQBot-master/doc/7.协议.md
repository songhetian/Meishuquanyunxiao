-------

5.获取登录token
-----
需要保存的几个参数，当然之前获得的cookie是必要的

	psessionid : 本次返回值
	client_id  : 步骤@4 自己生成
	ptwebqq    : 步骤@4 来源cookie
	uin        : 本次返回值
	vfwebqq    : 本次返回值

错误码：  

	{ retcode: 108, errmsg: '' } `缺少cookie`
	{ retcode: 103, errmsg: '' } `缺少cookie`
